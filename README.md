Getting Started  
Prerequisites  

In order to have this project up and running you will need:   

NodeJS 10+  
Setup  

clone this project  

Then run: npm install  

npm run dev-server, to run the project in your local machine.  

Then head to the localhost on port 8080 https://localhost:8080
